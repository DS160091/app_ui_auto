# 定位策略

## 优先级
1. **文本定位**：按钮、标题、Tab、提示文案
2. **文本 + 区域限定**：重复文案（多个同名按钮/卡片）
3. **唯一值精确定位**：单号/编号等唯一标识
4. **相对坐标**：无文本图标、底部导航
5. **图像特征**：悬浮按钮、返回/搜索图标

## 易误点
- <多个弹窗都有「确认」→ 结合弹窗标题/文案限定范围>
- <相似卡片 → 用唯一值定位>
- <同名文案 → 用父区域区分>
- <悬浮控件 → 忽略，别当业务控件>
